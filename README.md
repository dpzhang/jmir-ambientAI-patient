# Supplemental Material for "Ambient AI Scribe Implementation and Patient Experience Across Three Health Systems: A Multi-site Observational Study"

Authors: Dongping Zhang; Mukund Nakhate; Cheryl Stults; Heather Himelhoch; Urmila Ravichandran; Prsni Patel; Anthony Solomonides; Sien Deng; Joseph Wilcox; Naveed Rabbani; Sachin Shah; Veena Jones; Nirav Shah

Full Paper: Under review

## 1 About this repository

This repository contains the analysis code and generated figures associated with the manuscript **"Ambient AI Scribe Implementation and Patient Experience Across Three Health Systems: A Multi-site Observational Study."**

The underlying study data are not included because institutional policies restrict their public release. Fitted Bayesian model objects are also excluded because they are derived from the restricted data. Data may be shared upon reasonable request, subject to compliance with organizational agreements.

## 2 Content

The scripts are organized to document the analysis workflow. Researchers with authorized access to the data can reproduce the analyses by placing the required input file at `data/model.csv` and running the scripts in the order shown below. Fitted models produced by the first script are saved to `fitted_models/fitted_models.rds` and used by the remaining scripts.

The main analysis scripts should be run in the following order:

| Order | File                  | Description                                                                                                                                                                                                                                       |
| ----- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1     | `model_fit.R`         | Fits the Bayesian binomial models for the two outcome variables. The script fits the site-adjusted baseline, patient age composition, burnout context, and adoption scenario models under the default, stronger, and weaker prior specifications. |
| 2     | `model_analysis.R`    | Computes posterior pre/post contrasts, adjusted contextual contrasts, and scenario contrasts for the fitted models. It also provides exploratory summaries and plots for both study outcomes.                                                     |
| 3     | `model_diagnostics.R` | Evaluates model convergence and fit using R-hat, effective sample size, divergent transitions, and tree-depth summaries. It also creates MCMC trace plots and posterior predictive checks.                                                        |
| 4     | `prior_sensitivity.R` | Compares posterior coefficient estimates across the default, stronger, and weaker prior specifications and generates the prior-sensitivity figure.                                                                                                |
| 5     | `paper_figs.R`        | Produces the publication figures for the baseline, contextual, adoption, and scenario analyses. Generated figures are written to `figures/`.                                                                                                      |

Supporting materials are organized as follows:

| Directory        | Description                                                                                                                                                       |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `utils/`         | Contains helper functions used for model fitting, posterior contrasts, scenario analyses, diagnostics, prior-sensitivity analyses, and figure generation.         |
| `figures/`       | Contains figures generated by the analysis scripts.                                                                                                               |
| `data/`          | Expected location of `model.csv`. This directory and its contents are not included because the study data are subject to institutional data-sharing restrictions. |
| `fitted_models/` | Expected location of `fitted_models.rds`. This directory and its contents are not included because the fitted objects are derived from restricted study data.     |
